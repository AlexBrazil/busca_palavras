Como configurar uma nova atividade

Use este roteiro para preencher o arquivo data.json do caca-palavras.

Antes de comecar

  • Faca uma copia de seguranca de data.json.
  • Edite com um editor de texto simples e salve em formato UTF-8.
  • Ao terminar, salve e atualize index.html no navegador para testar.

Campos principais

mode

  • "palavra": mostra a lista com as mesmas palavras que estao escondidas.
  • "dica": mostra apenas dicas; cada item da lista precisa ter o texto em hint.
  • Qualquer outro valor cai no modo "palavra" automaticamente.

taskDescription

  • Mensagem inicial exibida acima da grade. Aceita texto simples e HTML basico.

wordList

Pode ser uma lista de objetos ou uma string com palavras separadas por virgula. Uso recomendado (principalmente para o modo "dica"):

"wordList": [
  { "word": "azul", "hint": "Cor do ceu em dia limpo" },
  { "word": "verde", "hint": "Mistura de azul com amarelo" }
]
  • word: texto a ser encontrado. O sistema converte para maiusculas automaticamente.
  • hint: dica opcional no modo "palavra" e obrigatoria no modo "dica".
  • Itens repetidos sao ignorados; deixe apenas uma ocorrencia.

behaviour

Controla como o tabuleiro e os botoes se comportam.

  • orientations: liga ou desliga direcoes. Defina true para permitir e false para bloquear. Opcoes: horizontal, horizontalBack (horizontal invertido), vertical, verticalUp (de baixo para cima), diagonal, diagonalBack, diagonalUp e diagonalUpBack.
  • fillPool: caracteres usados para completar as casas vazias. Use letras sem acento, por exemplo "ABCDEFGHIJKLMNOPQRSTUVWXYZ".
  • preferOverlap: mantenha true para incentivar cruzamentos entre as palavras.
  • showVocabulary: quando false, esconde a lista de palavras/dicas. O modo "dica" sempre reativa esse campo para mostrar as pistas.
  • enableShowSolution e enableRetry: exibem os botoes "Mostrar solucao" e "Repetir".
  • timeLimitSeconds: limite de tempo em segundos. Use zero ou remova para jogar sem cronometro regressivo.

instructions

Define o painel exibido antes do jogo comecar.

"instructions": {
  "enabled": true,
  "showOnLoad": true,
  "showOnRestart": false,
  "eyebrow": "Preparado?",
  "title": "Antes de comecar",
  "body": [
    "Leia a dica e encontre a palavra correspondente.",
    "Arraste o mouse ou o dedo para selecionar as letras."
  ],
  "steps": [
    "Use Corrigir para verificar o progresso.",
    "Clique em Mostrar solucao apenas se precisar."
  ],
  "startButtonLabel": "Iniciar desafio"
}
  • enabled: ativa/desativa o painel.
  • showOnLoad: determina se o painel abre ao carregar a pagina.
  • showOnRestart: reabre o painel quando o jogador clica em Repetir ou reinicia apos o tempo esgotado.
  • eyebrow e title: textos exibidos no topo.
  • body: lista de paragrafos (texto corrido).
  • steps: lista opcional exibida como marcadores.
  • startButtonLabel: texto do botao que fecha o painel e aciona o cronometro.

l10n

  • Permite trocar textos da interface (botoes, contadores e mensagens).
  • Mantenha os marcadores, como @found, @totalWords e @score, pois sao substituidos dinamicamente.

accessibility

  • Configura recursos extras. Atualmente ha suporte para o widget VLibras:
"accessibility": {
  "vlibras": { "desktop": true, "mobile": false }
}
  • Defina true para carregar o plugin na versao correspondente.

Checklist final

  • Verifique se todas as palavras possuem pelo menos duas letras.
  • No modo "dica", confirme que cada item tem hint.
  • Certifique-se de que o JSON esta valido (sem virgulas sobrando ou aspas faltando).
  • Teste a atividade marcando palavras certas e erradas, e valide os botoes.
